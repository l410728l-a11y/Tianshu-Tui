## 🐛 修复

### 子代理僵尸线程修复（严重）
- **根因**：delegate_task/delegate_batch 超时后，worker 继续运行为僵尸线程
- **表现**：调用子代理后主循环卡死，等待 worker 内部 180s 超时
- **修复**：将 AbortSignal 从工具流水线传递到 coordinator，超时时立即终止 worker
- 涉及文件：coordinator.ts, delegate-task.ts, delegate-batch.ts, tool-pipeline.ts, types.ts

### TUI 历史内容重复打印修复
- **根因**：`<Static>` 组件的 key 使用了滑动窗口首项 ID，每次窗口移动导致全量重绘
- **修复**：使用稳定的 `key="static-history"`，依靠子项 key 做增量渲染
- 消除了 agent 回复时屏幕向上翻滚、重复显示历史内容的问题

### AssistantMessage 高度溢出修复
- 修复 maxLines 与 viewport 比例的 min/max 反转
- 防止单条 assistant 消息占据过多屏幕空间

### 生成守卫防冻结
- generation guard 防止过期 run 冻结 streaming UI

### 交付门禁验证修复
- tool_invocation_failure 从 RED 降级为 YELLOW（不阻塞交付）
- targetFiles 覆盖机制：新验证结果替换旧结果

### API 超时扩展
